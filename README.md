# Vue.js Quiz App

A quiz app built with:
- Vue.js
- Vue Bootstrap
- Lodash
- [Open Trivia Database API](https://opentdb.com/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
